# fez-demo-75team

基于FEZ构建简单专题网站示例

> 本示例演示了基于FEZ构建简单官方网站，页面逻辑较简单，不再划分模块结构。

**下载示例**

- 将本示例下载到FEZ工程目录

```
cd fez工程目录

git clone https://github.com/furic-zhao/fez-demo-75team.git

fez
├── .babelrc
├── .editorconfig
├── .fez
├── .gitignore
├── README.md
├── fez.config.sample.js
├── gulpfile.babel.js
├── package.json
└── fez-demo-75team (项目目录)
```

**运行示例**

```
cd fez-demo-75team

gulp
```
